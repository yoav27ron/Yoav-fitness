# 💪 Yoav Fitness System — יואב פיטנס

> מערכת אישית מלאה לניהול אימונים, תזונה, שתיית מים והתקדמות לירידה במשקל.  
> **יעד: 100 ק"ג → 80 ק"ג**

---

## 🎯 סקירת המוצר

אפליקציית Web PWA בעברית, מותאמת לטלפון, עם שמירת נתונים ב-Firebase Firestore.  
המשתמש: יואב, גיל 35, גובה 176 ס"מ, משקל התחלתי ~100 ק"ג.

## 🍽️ מודול תזונה

### יעדים יומיים
| מדד | יעד |
|-----|-----|
| קלוריות | ~1,900 קל׳ |
| חלבון | 130g |
| פחמימה | 180g |
| שומן | 60g |

### חיפוש מוצרים
- **API:** Open Food Facts (`world.openfoodfacts.org`)
- תמיכה במוצרים ישראלים (תלמה, עלית, טרה, שטראוס וכו׳)
- תצוגה: תמונת מוצר + קלוריות ל-100g + מותג

### הוספת מוצר לארוחה — יחידות מדידה:
`מנה (100g)` / `גרם` / `מ"ל` / `יחידה` / `כוס (240ml)` / `כף (15g)` / `כפית (5g)` / `חופן (30g)`

### מבנה ארוחה שמורה
```json
{
  "name": "צלחת יוגורט",
  "time": "09:15",
  "items": [
    { "name": "יוגורט 3%", "qtyLabel": "1 כוס", "grams": 240, "cal": 144, "prot": 10, "carb": 14, "fat": 5 },
    { "name": "בננה", "qtyLabel": "1 יח׳", "grams": 120, "cal": 105, "prot": 1, "carb": 27, "fat": 0 }
  ],
  "cal": 249,
  "prot": 11,
  "carb": 41,
  "fat": 5
}
```

### העדפות תזונה של המשתמש
- אוהב: בשר בקר, עוף, דגים לבנים, טונה אדומה
- פחות אוהב: סלמון (אלא אם בסושי)
- ג׳אנק פוד: פעם בשבוע, עדיף שישי/שבת
- ירקות/פירות שלא אוהב: עגבניות טריות, פלפלים, קיווי, מנגו ועוד
- תפוחים: רק פינק ליידי או גרנד סמית׳

## 💧 מודול מים

- יעד יומי: **10–12 כוסות** (בגלל משקל 100 ק"ג)
- כל כוס = 250 מ"ל
- תזכורות: בוקר / אמצע יום / אחר הצהריים / ערב
- ביום אימון: +2 כוסות נוספות

---

## 📈 מודול התקדמות

- שקילה: פעם בשבוע, בוקר, לפני אוכל
- נקודת פתיחה: 100 ק"ג
- יעד: 80 ק"ג
- קצב ירידה מתוכנן: 0.5–1 ק"ג לשבוע
- זמן משוער: 7–9 חודשים

### מפת שלבים
| שלב | תקופה | מטרה |
|-----|--------|-------|
| 1 | שבועות 1–4 | הנחת בסיס, 2 אימונים |
| 2 | שבועות 5–10 | 3 אימונים, עלייה בנפח |
| 3 | חודש 3–5 | חיטוב + interval |
| 4 | חודש 6–9 | גמר — 80 ק"ג |

## 🏗️ ארכיטקטורה טכנית

### Frontend
- **שפה:** HTML5 + CSS3 + Vanilla JavaScript
- **פונטים:** Heebo (Google Fonts)
- **כיוון:** RTL (עברית)
- **עיצוב:** Dark theme — `#080c12` רקע, `#f5a623` accent

### Backend / Storage
- **Firebase Firestore** — אחסון ענן
- **Project ID:** `yoav-fitness--web-app`
- **Region:** europe-west3 (Frankfurt)
- **Fallback:** localStorage אם Firebase לא זמין

### מבנה נתונים ב-Firestore
```
collection: yoav/
  ├── workouts        → מערך אימונים
  ├── foodLog         → יומן אכילה (לפי תאריך)
  ├── weights         → היסטוריית משקל
  ├── waterCount      → ספירת כוסות מים יומית
  └── checks_{date}   → משימות יומיות
```

### קומפוננטים עיקריים
- **Hero** — כותרת עם פס התקדמות 100→80 ק"ג
- **Tab Bar** — sticky ניווט עליון
- **Check Item** — משימה עם כפתורי אישור/דחייה/ביטול
- **Training Card** — כרטיס אימון עם רשימת תרגילים
- **Workout Editor Modal** — עורך אימון מלא
- **Add Meal Modal** — הוספת ארוחה עם חיפוש מוצרים
- **Qty Modal** — בחירת כמות עם יחידות מדידה
- **Water Cups** — 12 כוסות לחיצות
- **Progress Bar** — פס התקדמות משקל
